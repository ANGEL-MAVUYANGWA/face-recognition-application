# 📌 Overview

The **Face Recognition System with AI Analysis** is a modern full-stack web application built using **Flask**, **OpenCV**, and **Artificial Intelligence concepts**.

This system allows users to upload images and receive:

- 👤 Face Detection
- 📊 Image Quality Analysis
- 🎨 Color & Composition Insights
- 📝 History Tracking
- 🔐 Secure Authentication

The project demonstrates practical implementation of:
- Computer Vision
- AI-powered Image Analysis
- Full-Stack Web Development
- User Authentication Systems

---

# ✨ Features

## 🔐 Authentication System
- User Registration
- Secure Login
- Password Hashing with bcrypt
- Session Management using Flask-Login

## 👤 Face Detection
- Detects faces using OpenCV Haar Cascades
- Works with single and multiple faces
- Draws bounding boxes around detected faces

## 📊 AI Image Quality Analysis
Analyze uploaded images for:
- Blur Detection
- Brightness Analysis
- Contrast Measurement
- Sharpness Evaluation

## 🎨 Color Analysis
- Detect dominant colors
- Analyze colorfulness
- Measure image vibrancy

## 📐 Composition Analysis
- Aspect ratio analysis
- Symmetry analysis
- Image balance evaluation

## 📝 History Tracking
- Stores previous image analyses
- View past uploaded images
- Persistent SQLite database storage

## 💻 Responsive Modern UI
- Beautiful Bootstrap 5 interface
- Mobile responsive design
- Interactive and clean user experience

# 🚀 Installation Guide

## 1️⃣ Clone Repository

```bash
git clone https://github.com/ANGEL-MAVUYANGWA/face-recognition-app.git
cd face-recognition-app
```

---

## 2️⃣ Create Virtual Environment

### Windows
```bash
python -m venv .venv
.venv\Scripts\activate
```

### macOS/Linux
```bash
python3 -m venv .venv
source .venv/bin/activate
```

---

## 3️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

---

## 4️⃣ Run Application

```bash
python app.py
```

---

## 5️⃣ Open Browser

```bash
http://127.0.0.1:5000
```

# 📸 How the System Works

1. User uploads an image 📤
2. OpenCV processes the image 🧠
3. Faces are detected automatically 👤
4. AI analyzes image quality 📊
5. Results are displayed visually 🎨
6. Analysis is saved to history 📝
